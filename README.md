"Genera el código de una aplicación en Flutter en un único archivo main.dart para un restaurante de sushi llamado 'Kinsui Express'. La aplicación debe cumplir con los siguientes requisitos estrictos:

Navegación por Rutas Nombradas: Configura el MaterialApp con las rutas: '/' (Inicio), '/productos', '/empleados', '/usuarios' y '/contacto'.

Widget Drawer Personalizado:

Encabezado (UserAccountsDrawerHeader): Fondo color negro (0xFF212121), nombre del negocio 'Kinsui Express' en color naranja, dirección 'Av. Sakura #456', teléfono '656-123-4567' y correo 'contacto@kinsui.com'.

Avatar: Una imagen circular cargada desde una URL de GitHub.

Opciones (ListTile): 5 opciones (Inicio, Productos, Empleados, Usuarios, Contacto) con iconos de Material Design y la acción Navigator.pushReplacementNamed para navegar.

Interfaz de las Páginas:

Página de Inicio (Home): Debe ser una pantalla de bienvenida estética con un título llamativo y una descripción corta del restaurante.

Páginas Secundarias: Cada una de las 4 opciones restantes debe tener su propio Scaffold, el Drawer funcional y, en el body, una imagen centrada de 200x200 px cargada desde un repositorio de GitHub (puedes usar identicons o logos de GitHub como placeholders).

Estilo Visual: Utiliza Material 3 con una paleta de colores basada en naranja (Colors.orange) y negro, evocando una temática de restaurante japonés moderno.

Código Limpio: Asegúrate de que el código sea válido para ejecutarse directamente en DartPad sin errores de compilación."
